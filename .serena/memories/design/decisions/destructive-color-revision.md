# Destructive Action Color Re-introduction
The user felt that the "Great Color Purge" made destructive buttons look "washed out" (grey).
I have proposed using a "Terracotta/Sumi Red" from the "Washi Palette" (used for habits) to restore visual weight.
This color is earthy and matte, aligning with the "Ink & Matte" style while providing the necessary danger semiotics.
